# GSMU
Небольшая учебная база ГСМ Учёт для учёта поступления и распределения топлива. 

Содержит справочники: Автомобили, Бензин, Ёмкости, Организации, Сотрудники.

Регистры накопления: БензинЁмкости, СебестоимостьБензина.

Документы: ПоступлениеБензина, ВыдачаБензина.

Общие модули: ПроверкаЗаполненияКлиент, ГСМУКлиент, ГСМУСервер, ОбменДаннымиСотрудники.

**Здесь вы можете скачать файлы .dt для ГСМ Учёт и Управление Торговлей (доработка и синхронизация изменений между Сотрудники <-> Водители)**
